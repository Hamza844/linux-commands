# Passenger

## check version
[How can I find Phusion Passenger version?](https://stackoverflow.com/questions/8174837/how-can-i-find-phusion-passenger-version)
```sh
passenger -v
```

Result:
```
Phusion Passenger(R) 6.0.12
```
